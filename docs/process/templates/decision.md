# 模板 [Decision] ADR — 架构决策记录

> 用途:记录**重大技术决策**("为什么用 X 库" / "为什么合并到 user 表" / "为什么走 JWT cookie 而非 OAuth")。
> 与 /change 的区别:决策**贯穿项目全程**,不像变更是一次性事件。
> 命名:`docs/decisions/NNNN-<short-title>.md`(NNNN 4 位 0 补,自续)
